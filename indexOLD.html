Contexte SACEM

La SACEM, société de gestion des droits d'auteur dans le secteur musical, a initié sa transformation digitale avec WSO2 API Manager en 2018.

L'objectif était de structurer et gouverner les échanges applicatifs au sein de la DSI, en passant d'une architecture on-premise à une plateforme hybride intégrant le cloud.

  • Gestion des droits d'auteur pour 200 000 membres
  • Transformation digitale de l'écosystème musical
  • Besoin d'une gouvernance API robuste

Parcours d'évolution

2018 - Lancement

Mise en place initiale de WSO2 API Manager en on-premise pour structurer les échanges applicatifs

2019 - Équipe transverse

Création d'une équipe dédiée à la gestion de la plateforme API

2020 - Haute disponibilité

Adoption d'une architecture haute disponibilité et début de l'hybridation cloud

2022 - Intégration SAFe

Alignement avec le framework agile à l'échelle et orientation vers des équipes produits

2024 - Self-service

Transition vers un modèle self-service avec gouvernance via workflows

2025 - Futur

Conteneurisation complète, montée de version vers WSO2 4.3 et intégration d'IA

Architecture et Innovations

Modèle Self-Service

Autonomie des équipes métiers avec gouvernance stricte via des workflows validés, intégrés aux pipelines CI/CD.

  • Publication simplifiée des API
  • Déploiement automatisé
  • Contrôle maintenu par la DSI

Plateforme Hybride

Architecture combinant on-premise et cloud pour une flexibilité maximale.

  • Haute disponibilité garantie
  • Scalabilité élastique
  • Sécurité renforcée
